KanlorOne Guard — 桌面一体化助手（屏幕使用 · 待办提醒 · 剪贴工具）

一、产品简介

KanlorOne Guard（乾御）是一款 桌面一体化助手工具，将屏幕使用管理、待办提醒、剪贴工具三大功能模块集成于同一托盘程序中，专为儿童护眼与时间控制以及成人专注工作两种场景设计。

程序常驻系统托盘，右键菜单即可快速操作；所有数据本地存储在 %LOCALAPPDATA%\KanlorAPP\KanlorOne_Guard\，不依赖云端，隐私安全有保障。试用期 30 天，结束后需注册授权才能继续使用。

三大模块：屏幕使用管理（用屏统计、休息提醒、工作模式、1 分钟预警）、待办提醒（四象限 / 列表 / 日历三模式、循环任务、智能识别）、剪贴管理（剪贴板整理、目录结构层级读取、颜色识别与转换）。

二、核心特色

屏幕 屏幕使用管理
  • 每日总时长限制（工作日 / 节假日分别设置）
  • 单次使用时长到达自动休息，1 分钟预警浮窗可拖动
  • 最近 7 天使用时长柱状图，自动识别工作日 / 节假日
  • 休息内容管理（名言警句 / 古诗词 / 小贴士），支持 JSON 导入导出
  • 工作模式：免密延时加时，按次数限制，替代暂停休息
待办 待办提醒
  • 四象限 / 列表 / 日历三种视图模式，农历与节假日显示
  • 子任务、循环任务（每天 / 工作日 / 每周 / 每月）、拖拽排序
  • 智能识别文本生成待办，支持腾讯会议信息识别
  • 置顶任务可生成浮动窗口，应用启动时自动显示
  • 自动备份（可配置间隔与保留天数），明色 / 暗色主题切换
剪贴 剪贴管理
  • 剪贴板自动整理：文本 / 图片 / 文件 / 文件夹全覆盖
  • 目录结构读取与变换（去除前置符 / 目录 / 文件 / 路径）
  • 目录层级设置：支持输入数字控制展开深度（0=全部，N=仅 N 层）
  • 4 种颜色格式自动识别与 HEX / RGB 双向转换
  • 系统取色 + 49 种预设颜色，色块竖向居中显示
✅ 核心优势
  • 托盘常驻：无主窗口，右键菜单快速操作，全局快捷键直达各模块
  • 隐私安全：所有数据本地存储，不依赖云端，不收集用户信息
  • 双场景适配：儿童用屏管理 + 成人专注工作两种场景一套工具搞定
  • 家长 PIN 保护：暂停守护、进入系统设置需 PIN 码验证

三、使用指南

1
启动程序：双击桌面 KanlorOne Guard 图标启动。程序在系统托盘（任务栏右下角）显示彩色图标，无主窗口。
2
呼出菜单：右键托盘图标弹出完整菜单（菜单项后显示当前快捷键）；双击托盘图标打开默认界面（默认是待办提醒，可在设置中修改）。
3
切换模块：通过托盘菜单或全局快捷键 Alt+1（屏幕使用）、Alt+2（待办管理）、Alt+3（剪贴管理）切换各功能面板。
4
设置 PIN：首次使用请在「系统设置」（Alt+0）中设置家长 PIN 码（≥4 位），用于保护家长配置和暂停守护操作。
5
配置限额：在系统设置中配置单次使用时长、休息时长、暂停时长、工作日 / 节假日总限额，以及工作模式（启用后到休息时间仅显示工作模式按钮）。
6
注册授权：试用期 30 天结束后，在「关于」（Alt+9）界面查看机器码，联系作者获取授权码完成注册。
托盘悬浮提示：鼠标悬停在托盘图标上会显示当前状态详情（剩余时长、待办数量、注册状态）。

四、功能详解

🛡️ 屏幕使用管理

通过托盘菜单或快捷键 Alt+1 打开屏幕使用面板，包含概览页、统计页、休息内容页三大子页。

📊 概览页
  • 四张状态卡片：当前状态、今日使用、本次会话、下次休息
  • 双进度条：今日使用进度（工作日 / 节假日分限额）、本次会话进度
  • 快捷操作：暂停 / 恢复、立即休息、设置（均带 PIN 验证）
  • 暂停横幅：守护暂停时显示倒计时，颜色醒目
  • 工作模式加时：会话时长已用完后显示"本次会话 X / Y 分钟"（Y 包含加时）
📈 统计页
  • 最近 7 天使用时长柱状图（自动识别工作日 / 节假日）
  • 每天使用量、限额参考线、最高值
  • 帮助用户直观了解一周使用趋势
📝 休息内容页
  • 三大分类：名言警句 / 古诗词 / 小贴士
  • 支持新增 / 编辑 / 删除休息内容
  • 支持 JSON 导入 / 导出，一键备份或迁移
  • 休息触发时随机抽取一条显示
⚙️ 休息与预警
  • 休息触发：单次使用时长达到设定值时自动进入休息界面，显示随机休息内容
  • 休息倒计时：支持配置倒计时秒数（默认 15 秒），屏锁前显示倒计时
  • 1 分钟预警：屏幕锁定前 1 分钟弹出醒目浮窗预警，浮窗可拖动到任意位置避免遮挡
  • 手动开始：可设置休息结束后需手动点击开始
工作模式 免密延时加时
  • 启用条件：在系统设置中启用工作模式，配置每次加时分钟数和每日次数上限
  • 触发时机：单次会话时长用完时，进入休息界面
  • 按钮替换：工作模式启用且有剩余次数时，休息界面仅显示"💼 工作模式加时 (已用/上限)"按钮，不显示"切换模式"（暂停休息）按钮
  • 点击后效果：状态切回 ACTIVE，本次会话从 0 开始重新计时，有效会话限制 = 基础限制 + 加时分钟
  • 次数用尽：达到每日次数上限后，恢复"切换模式"按钮，可正常进入休息
  • 显示规则：状态卡片显示"本次会话 X / Y 分钟"，Y 为基础限制 + 加时分钟数
🛑 暂停守护
  • 需输入家长 PIN 码
  • 暂停时长可在系统设置中配置（默认 10 分钟）
  • 暂停期间不计入使用时长，不触发休息
工作日 / 节假日判定：程序根据当前星期自动判断（周一至周五为工作日，周六周日为节假日）。工作日与节假日使用不同的总限额。

✨ 核心功能
  • 新建任务：双击空白区域或工具栏 ➕ 按钮；新建窗口打开时光标默认聚焦标题框，可直接输入
  • 智能识别：粘贴文本自动识别日期时间生成待办，支持腾讯会议信息识别（会议主题为标题，提前 5 分钟提醒）
  • 拖拽切换：在四象限模式下，拖动任务卡片可在象限间切换
  • 右键菜单：四象限和列表模式下，整行 / 整卡片右键可呼出快捷菜单（删除、置顶、完成、移动象限等）
  • 双击编辑：列表模式任意非交互区域双击即可编辑
  • 浮动窗口：置顶任务可生成浮动窗口，应用启动时自动显示
  • 子任务管理：每个任务可添加子任务清单（完成进度 完成数/总数）
  • 文本样式：支持 49 种颜色、粗体、斜体、字号自定义（任务卡片）
  • Emoji：新建任务随机分配 10 个待办相关彩色表情之一
  • 截止日期：支持按次数或按日期设定截止
  • 循环任务：支持每天 / 工作日 / 每周（可指定星期几）/ 每月（可指定几号），支持按次数或日期结束
  • 周期任务推进：点击"完成"仅完成本次轮次，自动推进到下一周期（日历视图按周期展开显示）
  • 贪睡（Snooze）：弹窗提醒时支持 5 / 10 / 30 分钟推迟
  • 导出 / 备份：支持手动导出 / 导入 JSON，支持自动备份（可配置间隔与保留天数）
  • 撤销 / 重做：Ctrl+Z 撤销 / Ctrl+Y 重做（最多 50 步）
  • 主题切换：支持明色 / 暗色两种主题
  • Shift+Enter：新建 / 编辑任务窗口中按 Shift+Enter 快速保存
🔔 提醒与声音
  • 待办到期自动弹窗提醒（含任务标题、截止时间、备注、子任务）
  • 弹窗时可选择完成 / 推迟 / 关闭，关闭后下次重新弹出
  • 声音同步：开启声音提醒时，弹窗出现与音乐播放同步
  • 声音停止：弹窗关闭（完成 / 推迟 / 关闭）时音乐立即停止
  • 提醒声音：支持自定义 WAV / WMA / MP3 文件，短音频自动循环播放（最长 5 秒）
  • 默认声音文件：%LOCALAPPDATA%\KanlorAPP\KanlorOne_Guard\KanlorOne_Guard.wav（放入即生效）
📅 日历视图进阶
  • 三种子视图：月（6×7 网格）/ 周（一周横向）/ 日（单日详情）
  • 农历显示：可勾选显示农历日期，农历初一自动加粗标红
  • 节假日标记：周末与法定节假日自动以浅红背景突出
  • 双击日期格：快速新建当日任务
  • 今日高亮：当前日期边框加粗强调
  • 任务徽标：每天最多显示 5 条任务，超出显示 +N
  • 周期任务展开：循环任务在日历视图中按周期展开，已完成轮次显示 ✅，当前轮次显示 ⌛，未来轮次显示 ⏳
浮动窗口：置顶任务的浮动窗口在应用启动时自动出现。关闭浮动窗口仅关闭窗口，不会取消置顶或标记完成。非置顶任务的浮动窗口在取消置顶、完成或删除时自动消失。
未注册受限：未注册时日历模式、导出 / 备份、智能识别等功能按钮灰色不可点击。

📋 剪贴管理

通过托盘菜单或快捷键 Alt+3 打开剪贴管理面板。

📡 剪贴板整理
  • 开启 / 关闭：点击按钮开启自动监听剪贴板
  • 文本：自动捕获剪贴板文本，显示到编辑框
  • 图片：自动保存到 桌面/KanlorOne/Clipboard_Images/，显示路径
  • 文件 / 文件夹：自动显示文件路径列表
  • 自动保存：所有内容自动保存到 桌面/KanlorOne/clipboard_auto_save.txt
  • 时间提示：可切换是否保留时间戳和类型标签
📁 目录结构工具
  • 读取目录：选择文件夹，生成标准 tree 格式树形结构（├── / └── / │）
  • 层级设置：输入数字控制展开深度，0 或空 = 全部层级，N = 仅展开到第 N 层（1=根的直接子项，2=孙子项，以此类推）
  • 去除前置符：移除 ├──、└──、│ 等树形符号
  • 去除目录：只保留文件路径行（以 / 结尾的视为目录）
  • 去除文件：只保留文件夹路径行
  • 去除路径：只保留文件 / 文件夹名称
  • 恢复：回退到上一步操作（最多 50 步历史）
  • 复制 / 导出：一键复制到剪贴板或导出为 TXT
🎨 颜色识别与转换

支持 4 种颜色格式的自动识别：

格式示例说明
HEX#FF5733 或 #F536 位或 3 位十六进制
RGB 带前缀rgb(255,87,51)支持中英文逗号和括号
仅括号(255,87,51)支持中英文逗号和括号
纯数字255,87,51三数组合，范围 0-255
  • 显示模式切换：全部 / HEX / RGB 三种模式，HEX 与 RGB 自动双向转换显示
  • 识别颜色：扫描编辑框文本，从上到下识别颜色，最多显示 12 个色块（有颜色时色块高度加倍突出）
  • 提取颜色：一键提取所有颜色，按当前显示格式输出，去除非颜色文本
  • 色块布局：颜色识别区域色块和文字竖向居中显示，文字与色块中心对齐
  • 点击色块可复制当前显示格式的颜色值
  • 色块区域支持水平滚动，避免色号文字被遮挡
🔧 系统取色

从屏幕任意位置取色（待办编辑器中的颜色面板提供 49 种预设颜色）。

剪贴板整理 vs 颜色识别：剪贴板整理开启时，颜色识别会自动进行，无需手动点击「识别颜色」按钮。

七、常见问题

Q：为什么听不到待办提醒声音？

检查系统设置中「启用待办声音提醒」是否勾选；确认 KanlorOne_Guard.wav 文件存在于数据目录；或自定义声音路径指向有效 WAV / WMA / MP3 文件。弹窗关闭时音乐会立即停止。

Q：开机自启设置了但没生效？

程序会在 %APPDATA%\Microsoft\Windows\Start Menu\Programs\Startup\ 创建快捷方式，并同步写入 StartupApproved\StartupFolder 注册表（12 字节，0x02=启用）。如果未出现在任务管理器启动项中，检查注册表是否被标记为 0x03（禁用）。

Q：新建任务窗口打开后可以直接输入吗？

可以。新建任务窗口打开时鼠标默认聚焦到标题输入框，可直接输入，Shift+Enter 快速保存。

Q：四象限视图任务卡片右侧有空白？

已优化：Canvas 内部 frame 宽度随 Canvas 同步自适应，任务卡片真正横向填满象限区域。

Q：工作模式启用后为什么不自动加时？

工作模式启用后，到休息时间不会自动加时，需手动点击"💼 工作模式加时"按钮。此时休息界面仅显示工作模式按钮和退出按钮，不显示"切换模式"（暂停休息）按钮。加时次数用尽后才恢复"切换模式"按钮。

Q：工作模式加时后为什么显示"本次会话 X / Y 分钟"？

工作模式加时后，Y 会包含加时分钟数（如基础 30 分钟 + 加时 30 分钟 = 60 分钟），X 从 0 开始重新计时。这样可直观看到加时后的有效会话限制。

Q：周期任务点击"完成"后为什么日期跳了 2 天？

v3.1 已修复此问题。原因是 get_due_todos() 在任务到期时过早推进了 due_time，complete_occurrence() 又推进了一次，导致双重推进。现在推进逻辑统一由 complete_occurrence() 处理。

Q：日历视图中周期任务如何显示？

周期任务在日历视图中按周期展开显示：已完成轮次显示 ✅（灰色），当前轮次显示 ⌛（过期显示 ⚠️ 黄色），未来轮次显示 ⏳（绿色）。

Q：试用期如何计算？

首次启动后 30 天为试用期。试用期内所有功能开放，试用期结束后未注册则功能受限。

Q：如何迁移数据到新电脑？

将 %LOCALAPPDATA%\KanlorAPP\KanlorOne_Guard\ 整个目录复制到新电脑相同位置即可。

Q：暂停守护后还会触发休息吗？

不会。暂停期间不计入使用时长，也不触发休息。暂停时长可在系统设置中配置（默认 10 分钟），需输入家长 PIN 码。

Q：未注册时哪些功能受限？

未注册时以下功能按钮灰色不可点击：颜色工具、剪贴板监控、待办导出、日历模式、智能识别、休息内容管理（新增 / 删除 / 导入导出，编辑不受限）。
